# Connector_Esp
Connectors used by Esprissifs ESP_PROG to connect both JTAG and programming port/uart. 

## 3D packages
To ensure the 3D packages load ensure the libs are place as follow

- ${KIPRJMOD}
  - /lib/
     - /Connector_Esp/ 
        -  Connector_Esp.lib
        -  Connector_Esp.3dshapes
        -  ...
     - /more-libs ... /
